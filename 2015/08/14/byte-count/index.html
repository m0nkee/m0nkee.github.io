值int a = 0x11223344存储方式

  • 大端字节序
地址 内容
低地址 0x11 a[0]
高地址 0x33 a[3]
  • 小端字节序
地址 内容
低地址 0x33 a[3]
高地址 0x11 a[0]

此处的a[0]仅表示a的高位即0x11其他依次类推

  • 调试

定义两个变量：

单步调试进去查看内存情况：

由上述内存情况分析可得：

  • 变量的存储是从高地址 –> 低地址，即m2的地址 < m1的地址
  • 该系统是小端存储的，如m1的高位0x11存放在高地址处
  • 注意数组的存放是从a[0]放在低地址处

所以,如果有如下共同体： 

union test {
    int i;
    char a[4];
}

union test t;
t.i = 0x11223344；
print("%x\n",t.a[3]);  
}

当运行的系统是小端字节序时，此时的a[3]的值就为0x11了。
